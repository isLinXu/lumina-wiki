# PyTorch

**PyTorch** 是一个基于 Python 的开源机器学习框架，广泛用于深度学习研究和开发。  
PyTorch 以动态计算图（即“定义即运行”）著称，支持灵活的模型构建和高效的 GPU 加速，同时提供丰富的库和工具，便于研究人员快速实验和迭代模型。  
它被广泛应用于自然语言处理、计算机视觉和强化学习等领域，许多前沿研究如 FlashAttention 都基于 PyTorch 进行实现和优化。
