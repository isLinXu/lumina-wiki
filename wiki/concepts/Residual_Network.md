# Residual Network

**Residual Network** 是一种通过引入残差学习框架来解决深度神经网络训练困难的模型。  
Residual Network 的核心思想是让网络学习输入与输出之间的残差映射，而非直接学习复杂的映射关系，从而缓解梯度消失问题，使网络可以更深层次地训练。这种结构通过跳跃连接（skip connection）实现信息的直接传递。  
该模型在图像识别领域广泛应用，尤其在 ImageNet 等大规模数据集上表现出色，推动了深度学习的发展，并启发了后续多种改进版本的提出。
