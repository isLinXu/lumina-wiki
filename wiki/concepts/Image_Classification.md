# Image Classification

**Image Classification** 是指将图像分配到预定义类别中的计算机视觉任务。  
它通过分析图像的像素信息，提取特征并判断其所属类别，是计算机视觉的基础技术之一。深度学习方法，如卷积神经网络，显著提升了分类的准确性和效率。  
典型应用包括人脸识别、医学影像分析和自动驾驶中的物体识别。在《Deep Residual Learning for Image Recognition》中，残差网络的提出进一步推动了图像分类性能的提升。
